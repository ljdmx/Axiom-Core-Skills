# Axiom Core v2.1: FSPC Pre-Flight Protocol (Phase 0)

## Phase 0A — Domain & Persona
> **Mandate**: Business Intelligence First.
- **Role × Permission Matrix**: Mandatory for multi-user systems.
- **Entity Threshold**: Model depth check (Min 5 entities).
- **User Journey Mapping**: Role-based flow diagrams.
- **PRD Sign-off**: Using `templates/product-spec.template.md`.

## Phase 0B — Architecture & Monetization
- **Tenant Isolation**: Shared Schema / Separate Schema / Separate DB.
- **Architecture Brief**: Tech stack decision document.
- **FinOps Estimation**: Monthly cost profiles based on RPS.
- **Value Capture**: Declare monetization strategy (SaaS/Marketplace/API).

## Phase 0C — Engineering Scaffold
- **Scaffold Logic**: CI/CD initialize + Husky + Commitlint.
- **Red Team Security Sync**: Automated vulnerability scan.
- **Spec-First API**: OpenAPI generation before code.

## Phase 0D — Phase-Locked State Machine
- **Backend First**: Schema + API verification before UI boot.
- **handshake logic**: Phase 1.5 Deep Asset Generation for 3D/Web3.
